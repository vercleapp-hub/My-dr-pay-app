# الـدكتور باي - ELDoctor Pay 🚀

تطبيق دفع إلكتروني متكامل مبني باستخدام **Kotlin Jetpack Compose** و **Supabase**. يوفر النظام بيئة آمنة للتجار والموزعين لإدارة العمليات المالية والخدمات الإلكترونية في مصر.

## 🌟 المميزات الرئيسية
- **نظام أدوار متعدد:** (أدمن، موزع، تاجر).
- **تتبع أمني متقدم:** تسجيل IP، بصمة الجهاز، والموقع الجغرافي لكل عملية.
- **إدارة الأرصدة:** نظام إيداعات بنكية مع موافقة يدوية من الإدارة.
- **استخراج تلقائي:** تحديد المحافظة تلقائياً من الرقم القومي المصري.
- **دعم فني:** نظام تذاكر متكامل لمتابعة شكاوى الفواتير.
- **طباعة حرارية:** دعم الطباعة عبر البلوتوث لإيصالات العمليات.

## 🛠 التكنولوجيا المستخدمة
- **Android:** Kotlin, Jetpack Compose, Coroutines.
- **Backend:** Supabase (Auth, Database, Storage, Realtime).
- **Deployment:** GitHub & Vercel.

## 📊 هيكل قاعدة البيانات
تم ضبط قاعدة البيانات على توقيت القاهرة (+20) وتشمل جداول:
1. `profiles`: بيانات المستخدمين والأجهزة.
2. `transactions`: سجل العمليات المالية.
3. `deposits`: طلبات شحن الرصيد.
4. `services`: خدمات الدفع (e-Misr).

## 🚀 التشغيل
1. قم بربط تطبيق الأندرويد بمفاتيح Supabase في ملف `SupabaseClient.kt`.
2. قم بتشغيل سكربت `supabase_schema.sql` في محرر SQL الخاص بـ Supabase.
3. ابدأ في استقبال طلبات تسجيل التجار!
