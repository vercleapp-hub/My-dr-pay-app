# 🚀 ELDoctor Pay - Fintech Solution

**منظومة الدفع الإلكتروني المتكاملة (أندرويد + ويب)**

نظام "ELDoctor Pay" هو حل تقني متكامل لإدارة عمليات الدفع الإلكتروني، شحن الرصيد، وفواتير المرافق، مصمم خصيصاً للتجار مع ربط مباشر بنظام Supabase و e-misr.

## 🌟 المميزات الرئيسية
- **نظام Dashboard ذكي:** عرض الرصيد لحظياً وتتبع العمليات.
- **Dynamic Services:** دعم الخدمات المتغيرة (شحن موبايل، فواتير كهرباء، غاز، مياه) عبر نماذج ديناميكية.
- **أمان فائق:** تسجيل دخول ببصمة الجهاز (Device Fingerprint) وتتبع الموقع الجغرافي (GPS).
- **إدارة المحفظة:** نظام إيداع رصيد متكامل مع مراجعة الإدارة.
- **طباعة الإيصالات:** دعم الطابعات الحرارية (Thermal Printers) عبر الأندرويد.

## 🛠️ التقنيات المستخدمة
- **Android:** Kotlin, Jetpack Compose, Retrofit, Coroutines.
- **Web:** Vanilla JS, Supabase JS SDK, CSS3 (Modern UI).
- **Backend:** Supabase (Auth, PostgreSQL, RLS Policies).
- **API Connectivity:** e-misr Integrated.

## 🚀 التشغيل السريع
1. قم بربط المشروع بـ Supabase Project الخاص بك عبر `SupabaseClient.kt`.
2. قم بتشغيل واجهة الويب عبر فتح `index.html`.
3. لتشغيل الأندرويد، استخدم Android Studio (Giraffe+) وقم ببناء الـ APK.
